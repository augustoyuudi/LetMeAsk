## Getting Started

### Prerequisites

* [npm](https://nodejs.org/en/) / [yarn](https://yarnpkg.com/)
  

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:augustoyuudi/nlw-06-trilha-reactjs.git
   ```
2. Install NPM packages
   ```sh
   yarn
   ```
3. Start development server
   ```JS
   yarn start
   ```

### Build to production

1. Build into folder /build
   ```sh
   yarn build
   ```
